## Installation

### From Chrome Web Store (Recommended)
1. Visit the [SummaTube Chrome Extension page](https://chrome.google.com/webstore) on the Chrome Web Store
2. Click "Add to Chrome"
3. Confirm the installation when prompted

### Manual Installation (Development)
1. Download or clone this repository
2. Open Chrome and navigate to `chrome://extensions/`
3. Enable "Developer mode" in the top right corner
4. Click "Load unpacked" and select the `chrome-ext/` folder
5. The extension will be loaded and ready to use

### OpenAI API Key Setup

**Required for AI Summaries**: You need an OpenAI API key to generate video summaries.

1. Visit [OpenAI Platform](https://platform.openai.com/api-keys)
2. Sign up or log in to your account
3. Navigate to API Keys section
4. Click "Create new secret key"
5. Copy the generated API key
6. In SummaTube, click the "üîë Set API Key" button and paste your key

**Note**: Your API key is stored locally in your browser and never sent to external servers except OpenAI.

### AI Summarization

## Permissions

SummaTube requires the following permissions:

- **`activeTab`**: Interact with YouTube pages to extract transcripts
- **`storage`**: Store your OpenAI API key and cache summaries locally
- **`host_permissions`**: Connect to `https://api.openai.com/` for AI summarization

## Privacy & Security

- **API Key Storage**: Your OpenAI API key is stored locally in your browser's storage
- **Direct API Calls**: Summaries are generated by calling OpenAI directly from your browser

## Troubleshooting

### Extension Not Appearing
- **Refresh the page**: Sometimes YouTube's dynamic loading requires a refresh
- **Check video compatibility**: Make sure the video has captions/transcripts available
- **Verify installation**: Ensure the extension is enabled in `chrome://extensions/`

### Transcript Not Loading
- **Wait a moment**: Transcript extraction takes 1-2 seconds
- **Check video language**: Currently supports videos with available captions
- **Try refreshing**: YouTube's interface may have changed

### AI Summary Issues
- **Check internet connection**: Summaries require API access
- **Verify API key**: Ensure your OpenAI API key is set and valid
- **Check OpenAI account**: Make sure you have credits in your OpenAI account
- **Wait for processing**: AI summarization takes a few seconds
- **Retry if failed**: Network issues or API limits can cause temporary failures

### API Key Issues
- **Key not saved**: Try refreshing the page after setting your API key
- **Invalid key**: Verify your API key from OpenAI Platform
- **Rate limits**: OpenAI has usage limits depending on your plan
- **Security**: Your API key is stored locally and only used for OpenAI requests

### Performance Issues
- **Close other tabs**: Multiple YouTube tabs may impact performance
- **Clear browser cache**: Old cached data can cause issues
- **Restart browser**: Sometimes a fresh start resolves problems
